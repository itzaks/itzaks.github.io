# What is Omform?

A new kind of animation app. It is based on shape morphing, which means you draw two forms, and the app animates between these two shapes by means of morphing.

# Loop based

Based on a lot of music production application, this app works with loops. You set a loop duration and the speed of the animation are then relative to this length, it can be set to different fractions of the total speed. This kind of fraction based timing system makes everything very rhythmic.

# Easings

There are a lot of different easings to work with in the app, that makes the animation move in different motions, quick, smooth, bouncy, glitchy, and so on.

# Export as SVG

The final creation can be exported as SVG to be used on the web. Exporting as x264 video is also available.

# Instantanious

Draw as the animations are playing. Change a color and it reflects that change immideatly.

# Active development

This app is active development and eagerly listens to feedback from the community.
